# Enterprise Overview

Solvela Enterprise adds multi-tenant organizations to the payment gateway. A single gateway deployment can host many independent organizations, each with their own teams, API keys, budgets, and audit history.

## Authentication

Enterprise endpoints accept two forms of authentication:

| Method | Header | Notes |
|--------|--------|-------|
| Admin token | `Authorization: Bearer <admin-token>` | Full access to all orgs |
| Org API key | `Authorization: Bearer solvela_k_...` | Scoped to a single org |

Admin-only actions (such as creating an organization) require the admin token. All other endpoints accept either form.
